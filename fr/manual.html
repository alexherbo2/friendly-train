Manuel Shortcuts

Disponible à partir de cette version de Chrome.

Guide d’utilisation

Alt+Espace (Ctrl+Espace sous Windows) est le raccourci principal. Utilisez-le pour ouvrir le menu de l’extension—aka mode « Vim ». Appuyez sur Ctrl+KeyF pour rechercher des commandes ou la touche d’échappement « Escape » pour quitter.

Si vous avez besoin d’explications sur les raccourcis—Faites un clic droit sur l’icône de l’extension et sélectionnez « Documentation ».

Les raccourcis clavier sont entièrement personnalisables.

Palette de commandes

La palette de commandes est le principal moyen de rechercher des fonctionnalités dans Shortcuts.

  1. Pour ouvrir la palette de commandes—Appuyez soit sur la touche Slash, Ctrl+KeyF ou cliquez dans le champ de recherche du menu de l’extension.
  2. Dans le champ de recherche, saisissez l’objet de votre recherche—les résultats s’affichent pendant votre saisie.
  3. Choisissez une suggestion et appuyez sur la touche d’entrée « Enter » pour activer la sélection.

Configurer les raccourcis clavier

Accédez à chrome://extensions/shortcuts pour configurer les raccourcis globaux. Par exemple, vous pouvez assigner Ctrl+Y pour ouvrir un nouvel onglet à droite. Vous pouvez également assigner des raccourcis pour dupliquer un onglet, copier le lien d’une page et plus encore. Voir la liste des actions pour Chrome.

Vous pouvez également configurer les raccourcis du menu de l’extension en important et exportant les paramètres dans la page des options—Faites un clic droit sur l’icône de l’extension et sélectionnez « Options ».

En pratique ?

Alt+J et Alt+K sont les deux raccourcis que j’ai configuré pour accéder au menu Shortcuts—dans l’éditeur de raccourcis « Activer l’extension »—et cliquer avec Link Hints—dans les options « Raccourcis clavier ».

Pour compléter les raccourcis clavier fournis par défaut, j’ai également assigné Ctrl+Y pour ouvrir un nouvel onglet à droite, Ctrl+B pour dupliquer un onglet, Alt+W pour effectuer une recherche du terme sélectionné, Alt+Y pour copier le lien d’une page, et Alt+I pour passer au champ de saisie suivant.

Commandes

Les commandes sont des actions qui peuvent être effectuées via le clavier ou la souris—dans le menu de l’extension.

À propos des raccourcis clavier

  • ⌃ représente la touche Ctrl sur votre clavier.
  • ⌥ représente la touche Alt ou Opt sur un clavier Mac.
  • ⇧ représente la touche Maj sur votre clavier.
  • ⌘ représente la touche Windows ou Cmd sur un clavier Mac.

Les raccourcis clavier du menu de l’extension sont définis avec des touches physiques. Ces touches sont présentées ci-dessous. Les touches bleues sont présentes sur tous les claviers standards, tandis que les touches vertes ne sont disponibles que sur certains claviers.

Pour obtenir la liste complète des touches physiques, consultez l’article Keyboard Event code Value Tables (en anglais). Vous y trouverez la liste des touches physiques et des symboles utilisés dans les raccourcis clavier—qui sont affichés dans les commandes du menu de l’extension.

Astuce : Vous pouvez cliquer sur les touches roses—qui représentent une touche physique—pour voir la touche à appuyer (p. ex., cliquer sur KeyQ affichera « a » sur un clavier français).

Trucs et astuces

  • Pour fermer les onglets sur la droite—Appuyez sur Tab, suivi de Maj+KeyS pour sélectionner les onglets sur la droite, puis KeyX pour fermer la sélection.
  • Pour fermer les autres onglets—Appuyez sur la touche de début « Home » pour déplacer l’onglet que vous souhaitez conserver en première position, puis fermez les onglets sur la droite.
  • Pour déplacer un groupe d’onglets—Appuyez sur KeyA pour sélectionner les onglets du même groupe, puis les touches fléchées « ArrowLeft » ou « ArrowRight » pour déplacer la sélection vers la gauche ou la droite.
  • Pour fusionner des fenêtres—Appuyez sur Maj+KeyA pour sélectionner tous les onglets, puis la flèche vers le bas « Down » pour déplacer la sélection vers la fenêtre précédente.

Pour découvrir plus d’astuces, consultez l’article Astuces Chrome.
